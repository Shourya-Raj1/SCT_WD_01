---

## ‚ú® Key Features

* **Fixed Navigation:** The menu remains accessible at the top of the viewport regardless of scroll position.
* **Dynamic Scroll Styling:** Utilizing JavaScript to detect scroll depth and trigger CSS class changes (e.g., background color shift, shadow depth).
* **Interactive Hover Effects:** Smooth transitions on menu items to improve navigation clarity.
* **Fully Responsive:** A "mobile-first" approach ensures the layout adapts seamlessly to smartphones, tablets, and desktops.
